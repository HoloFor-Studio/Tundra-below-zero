# Tundra-below-zero

## Описание

### Коммерческое описание
1. Название: Tundra: below zero
2. Жанр: Кооперативное хардкорное выживание
3. Платформа: ПК, Портативные консоли, Консоли
4. Игровой движок: Unity 3D
5. Возраст целевой аудитории: 15+
6. Возрастной рейтинг: 18+

### Game-Дизайн

#### 1.Крючок (Elevator Pitch)
- Tundra: below zero — это одиночное/кооперативное хардкорное выживание с элементами RPG. где игрок(-и) должны адаптировать свою экипировку и оружие в постапокалиптическом ледяном аду, чтобы помочь своему дому несмотря на адские погодные условия, животных, мутантов и людей которые озверели

#### 2.Основные Столпы (Core Pillars)
1. Тактическое выживание: Каждое решение (экипировка, маршрут) имеет вес и последствия.
2. Постояная угроза: Мир активно враждебен (погода, мутанты, люди).
3. Осмысленный кооп: Роли дополняют друг друга; выжить в одиночку значительно сложнее.
4. Характеристики: Игрок должен нарастащивать свои характеристи чтобы быстрее лечится или перезаряжать оружие

#### 3.Основной геймплейный цикл (Core Loop)
- Планирование => Исследование => Угроза => Обратная связь.

#### 4.Ключевые механики (Key Features)
| Механика | Суть | Как влияет на игрока | Пример из игры |
| ----------- | ----------- | ----------- | ----------- |
| Слоёная экипировка   | Комбинация одежны под нужные условия  | Тактический выбор между защитой, подвижностью и скрытностью.  | Для мороза в -50 нужно закрыть всё тело, для -5 хватит обычой куртки |
| Локализованые ранения    | Каждая конечность имеет своё ХП и в случае повреждения наносит эффекты на игрока| Ранения конкретно ограничивают возможности (хромота, дрожь в руках)  | Пуля в ногу => хромота; пуля в руку => снижение точности. |
| Характеристики    | Характеристики учавствуют в разных аспектах выживания начиная переносимостью холода, заканчивая обращение с оружием | Игрок по мере всего выживания развивается и становится сильнее | С развитой переносимостью к холоду можно в -10 в обычной ветровке |
| Крафт | Позволяет в полевых условиях получать нужные ресурсы | Игрок может самостоятельно получать разные предметы такие как бинты, шины, фильтры для противогаза и одежду| Игрок рвёт футболку на бинты чтобы перевязать рану |

#### 5.Описание Мира (World Overview)
1. Сеттинг: Постапокалептический мир заметёный снегом и льдом, после климатической катастрофы, вызваниый материалом Криомин.
2. Ключевые локации:
   - На поверхности: Городская местность с хрущёвками и магазинами, ТЦ "Курорт", Деревня, Лесная чаща, Промышленая зона с заводами и складами.
   - Под землёй: Подвалы, Канализация, Метро, Лаборатории, подземные фабрики.
3. Цветовая паллитра: Грязно-белые тона с серыми оттенками, акцент на тёплых тонах(кровь, огонь, места прибывания людей).

#### 6.Структура Прогресии (Progression Structure)
- Акцент не на характеристики, а на количество экипировки для составления адаптивных сетапов.
- Цена прогресса: Потеря ресурсов, всё труднее получать новую экипировку.
